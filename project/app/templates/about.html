Information about this application

This application was built to demonstrate the use of HTMX. HTMX is a tool that embeds behavior into HTML to produce SPA (Single Page Applications) very much like those created with tools like React or Angular. The difference is the applications can be created completely server side.


For this application I built on the FastAPI application I created as a presentation about FastAPI, which you can find here. That application wrapped a REST interface around the Chinook database that's part of the SQLite Tutorial. This application wraps a UI around that functionality.


This application is written using Python, but the HTMX framework is programming language agnostic. Any language that can produce HTML web pages can use the tool to add amazing and interesting interactivity to the app.


Shameless Disclaimer. This is the first HTMX application I've written. That means I barely scratched the surface of what it can do and what features I took advantage of. Given that, I think it's an okay application to look at for anyone starting to consider using HTMX.
